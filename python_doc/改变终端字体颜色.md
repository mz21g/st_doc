#### python terminal color

> **The most basic terminals have a set of 8 different colors:**
>
> - **Black**: \u001b[30m.
> - **Red**: \u001b[31m.
> - **Green**: \u001b[32m.
> - Yellow: \u001b[33m.
> - **Blue**: \u001b[34m.
> - Magenta: \u001b[35m.
> - Cyan: \u001b[36m.
> - White: \u001b[37m
